# Simple Data Engineering Project

A lightweight data engineering project for ETL operations.

## Project Structure

```
De_Project_test/
├── data/              # Raw data files
├── scripts/           # Python ETL scripts
├── sql/               # SQL queries
├── config/            # Configuration files
├── logs/              # Log files
├── output/            # Processed output
├── requirements.txt   # Python dependencies
└── README.md         # This file
```

## Quick Start

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run ETL script:
   ```bash
   python scripts/etl.py
   ```

3. Check output in `output/` folder
